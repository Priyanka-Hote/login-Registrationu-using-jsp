Employee Login and Registration Web App


This project is a web application for employee login and registration. It allows employees to create an account, log in, and manage their profile information. The application is built using Java for the server-side logic, MySQL for the database, and Apache Tomcat version 9 as the web server.


Features


- Employee Registration: New employees can create an account by providing their name, date of birth, gender, address, city, state, email, and password.


- Employee Login: Registered employees can log in to their accounts using their email and password.




Technologies Used


- Java: The server-side logic is written in Java, making use of Java Servlets and JavaServer Pages (JSP).


- MySQL: The project uses MySQL as the relational database to store employee information securely.


- Apache Tomcat 9: Apache Tomcat is used as the web server to deploy and run the web application.
